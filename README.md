# React + TypeScript + Vite

## 运行流程

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 页面操作说明

### 1. 基础操作
- **旋转视角**：鼠标左键按住拖动
- **平移视角**：鼠标右键按住拖动
- **缩放视角**：鼠标滚轮滚动

### 2. 搭建管架
- **添加管件**：点击顶部工具栏的「添加20cm管」或「添加40cm管」按钮。
- **放置管件**：移动鼠标选择位置，点击确认起点，再次点击或按 `Enter` 键确认放置。
- **切换方向**：在放置过程中，按 `R` 键或点击「切换方向」按钮可改变管件延伸方向。
- **取消操作**：按 `Esc` 键取消当前放置状态。

### 3. 编辑与管理
- **删除管件**：点击选中已放置的管件（选中后会变色），按 `Delete` 或 `Backspace` 键删除。
- **一键生成**：点击「一键生成结构」，输入手头拥有的管件数量，系统会自动生成推荐的结构。
- **保存截图**：点击「一键截图」可以下载当前设计图及物料清单图片。
